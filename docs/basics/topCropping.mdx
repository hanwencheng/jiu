# 顶层酵母提取

艾尔酵母是上层发酵的酵母，他们亲水的表层会使酵母附着在发酵产生的二氧化碳上，从而上升到液体的表层。

上层提取酵母最重要的一点就是无菌环境

## 优点

上层提取的酵母的质量比底层的要好：

1. 他们提取的时候是在活性最强的时候
2. 与底层细菌，啤酒花粉等的杂质相分离
3. 底层有更高的水压
4. 酿酒师的人工基因选择使酵母更倾向于浮于表面

## 提取时间
发酵开始之后的 12 小时 - 3，4天

## 要点
选择大的，平的，大开口的发酵罐
创造无菌的环境
对于顶层发酵很强烈的酵母，留有足够的上层空间
上层最表层有很多的蛋白质，在提取的时候我们希望去除尽量多的蛋白质，所以这一层最好给去掉
第二瓢和第三瓢含有活性最强的酵母
理论上任何器具都行，比如光滑表面的不锈钢勺
如果用勺子的话，不要全部打开盖子，尽量避免与外界空气的接触，确保附近没有大的空气流动
